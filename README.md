# 💈 Alabama Barbers - Sistema de Gerenciamento

Sistema web desenvolvido para gerenciamento de uma barbearia, com foco em **agendamento de serviços, controle de clientes e organização da rotina de atendimentos**.

---

## 📌 Sobre o Projeto

O **Alabama Barbers** é uma solução digital criada para substituir processos manuais (como anotações em papel ou mensagens informais), proporcionando:

* Maior organização
* Redução de erros
* Melhor experiência para clientes e barbeiros

O sistema permite que clientes agendem horários facilmente e que o barbeiro gerencie sua agenda de forma eficiente. 

---

## 🎯 Objetivos

* Facilitar o agendamento de serviços
* Organizar a agenda da barbearia
* Centralizar o controle de clientes
* Reduzir conflitos de horários
* Melhorar a experiência do usuário 

---

## 👤 Público-Alvo

* Barbearias de pequeno e médio porte
* Profissionais autônomos da área de estética
* Negócios que utilizam agendamento de serviços

---

## ⚙️ Funcionalidades

### 👥 Área do Cliente

* Visualizar serviços disponíveis
* Escolher data e horário
* Realizar agendamento online
* Receber confirmação do agendamento
* Consultar informações da barbearia

### 🛠️ Painel Administrativo

* Visualizar agenda de atendimentos
* Gerenciar horários disponíveis
* Acompanhar agendamentos
* Atualizar ou cancelar atendimentos
* Organizar rotina de trabalho 

---

## 🖥️ Interface

O sistema conta com:

* Landing page moderna inspirada em sites de barbearia
* Tela de agendamento com seleção de data e horário
* Tela de confirmação de agendamento

Essas interfaces foram pensadas para serem simples, intuitivas e acessíveis.

---

## 🧠 Metodologia

O projeto foi desenvolvido utilizando a metodologia ágil **Kanban**, com organização via Trello, permitindo:

* Visualização do fluxo de tarefas
* Priorização de demandas
* Acompanhamento do progresso em tempo real
